Navigationsbar

Appen gør brug af en navigationsbar i bunden af skærmen som indeholder href's til home, elpriser, tips & råd og profil. Når et ikon er orange indikerer det hvilken side man er på i øjeblikket. I nedenståendes tilfælde så er man på home-siden:

Navigationsbar

Html


            <div class="footer">
              <div class="icons">
                <a href="home.html"> <i class="fa-solid fa-house fa-xl">/i> </a>
                <a href="elpriser.html"> <i class="fa-solid fa-bolt fa-xl"></i> </a>
                <a href="#" id="active"> <i class="fa-solid fa-lightbulb fa-xl"></i> </a>
                <a href="#"> <i class="fa-solid fa-user fa-xl"></i> </a>
              </div>
            </div>

            <nav class="navtop" id="navtop">
              <ul>
                <li><a href="#"><img src="images/backarrowicon.svg" alt="Tilbage"></a></li>
                <li><a href="#"><img src="images/homeicon.svg" alt="Hjem"></a></li>
              </ul>
            </nav>

Typografi

Typografien er inspireret ud fra nunito som fremgår af Schneider Electrics style guide, der er dog valgt at anvende Europa som typografi i stedet, idet den ligner og er universel for alle enheder:

Kode i css


            * {
              font-family: Europa;
          }

            h1 {
              font-weight: bold;
              color: #FFF;
            }

            p {
              color: #000;
            }

Knapper

Størstedelen af knapper i appen, har runde kanter med grøn, sort eller hvid tekst.

HTML

            <a href="#"><div class="laesmere">Find løsninger</div></a>

CSS


            .laesmere {
              display: inline-block;
              background-color: #FFF;
              border: 1px solid black;
              text-align: center;
              color:#6BCA66;

              padding: 20px;
              border-radius: 30px;
              position: relative;
              font-weight: bold;
